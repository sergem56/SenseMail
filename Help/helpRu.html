Введение

Программа предназначена для безопасного обмена почтовыми сообщениями. Сообщения шифруются стандартными средствами операционной системы по алгоритму AES256. В качестве ключа выступает вводимый пользователем пин-код. Для каждого сообщения можно назначить свой собственный пин-код, который необходимо сообщить адресату по другим каналам (телефон, личная встреча и т.д.). Пин-код - это строка от одного знака до любого другого разумного количества знаков. Ключ шифрования получается из пин-кода стандартным алгоритмом "медленного хеша" (PBKDF2) и занимает около 1 секунды.
Можно обменяться с адресатом предварительно созданными сертификатами, что значительно повышает стойкость шифрования сообщений. При отправке сообщения автоматически проверяется наличие сертификата и если он есть, сообщение шифруется им. Сертификат представляет собой случайную 256-битную последовательность, которая хранится на устройстве зашифрованная общим пин-кодом и отдельным ключом, получаемым из пароля сертификата 2-х секундным "медленным хешем".


Адресная книга, настройки программы и безопасная галерея хранятся на устройстве зашифрованные общим пин-кодом, задаваемым в настройках.


Заголовок сообщения шифруется алгоритмом AES, но в качестве ключа выступает адрес отправителя, поэтому следует считать, что тема письма не является секретом.


Рекомендуется использовать специально выделенный почтовый ящик на сервере gmail.com, так как этот сервис предоставляет наиболее полную функциональность (помеченные сообщения, список папок и др.)


В целях безопасности:

  • По окончании работы с программой всегда закрывайте ее кнопкой выключения, так как в противном случае операционная система автоматически сохраняет введенный пин-код и скриншот последнего экрана
  • Выбирайте пин-код разумно - это единственная защита от прочтения ваших сообщений, если устройство попадет в руки злоумышленников. Устанавливайте как минимум 6-символьный код. 8 символов сделают перебор паролей экономически невыгодным в подавляющем большинстве случаев.
  • Программа не хранит никаких открытых данных локально, за исключением вложений, которые были сохранены пользователем в стандартную галерею устройства.
  • Тема письма считается открытой информацией.


Работа с программой

Список сообщений

При запуске программы открывается главное окно со списком сообщений. Для получения сообщений необходимо нажать на кнопку , либо нажать на строчку таблицы "Загрузить еще". Сообщения получаются с сервера блоками по 10 штук. Чтобы получить следующий блок, нужно нажать на строчку таблицы "Получить следующие сообщения".
Во время получения сообщений все операции с сообщениями заблокированы.


Для просмотра других доступных папок нажмите кнопку


Чтобы удалить сообщение смахните его влево - появится кнопка "Удалить".


Чтобы пометить или снять пометку с сообщения нажмите на кнопку или слева от него. Все помеченные сообщения доступны по кнопке


Чтобы просмотреть сохраненные заметки нажмите кнопку


Чтобы просмотреть сохраненные изображения нажмите кнопку


Чтение сообщения

Для чтения сообщения нужно нажать на него. Появится окно, предлагающее ввести пин-код. Если у сообщения есть вложения, то они будут загружены. Вложения будут показаны в виде миниатюр. Для просмотра вложений нажмите на миниатюру. В появившемся окне можно увеличивать изображение и сохранить его в стандартной или безопасной галерее.


Чтобы ответить на сообщение нажмите кнопку "Reply". В открывшемся окне заполните все графы и нажмите кнопку "Отправить".

ВАЖНО: Для безопасности, при проблемах со связью, когда сообщение не может быть отправлено сейчас, его нельзя сохранить, т.к. на устройстве запрещено хранить информацию в открытом виде.


Настройки программы

  • SMTP server, IMAP server, SMTP port - настройки, получаемые у провайдера почтовых услуг.
  • User name, password - данные для входа в почтовый ящик. Требуется указать полный адрес электронной почты, включая часть после @
  • User nick - имя, которое будет указано в сообщении. Оно же будет указано в первой строчке в списке сообщений.
  • Пин-код - пароль, запрашиваемый при старте программы. Его можно изменить, при этом будут перешифрованы контакты, настройки и файлы, сохраненные в безопасной галерее.
  • Сжатие JPG - при передаче вложения изображение будет сжато с указанным уровнем сжатия. Значение 1 - это максимальное качество. При этом размер файла, полученного со встроенной камеры Iphone 4S, будет 6-8 Мб. При уровне сжатия 0.6 та же самая фотография будет иметь размер около 1.5 Мб при незначительном снижении качества.

Контакты

В программе существует своя адресная книга. Она хранится локально. Список контактов открывается в окне нового сообщения при нажатии кнопки "+" рядом с полем "Кому".


Известные ограничения

  • Число вложений рекомендуется ограничить 10, так как возможны проблемы с памятью на iPod-ах.
