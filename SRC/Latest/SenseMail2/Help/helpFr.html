Qu'est-ce que SenseMail?

SenseMail est une application qui peut se connecter à votre fournisseur de services de messagerie préféré et protéger vos communications.


Tout d'abord, la protection est la combinaison de pointe de compression, de cryptage et de signature - l'algorithme AES-256 standard pour le cryptage, gzip pour la compression, HMAC et AONT pour la signature et la vérification.


Deuxièmement, nous utilisons PBKDF-SHA512 pour dériver la clé de chiffrement des données initiales. Le nombre de tours varie au hasard de près de 300K à près de 800K pour être sûr qu'il est suffisamment grand pour rendre le forçage brut impossible.


Le dernier mais non le moindre est le moyen d'obtenir les données initiales pour la clé de chiffrement. Il y a trois possibilités:


1. Un mot de passe. Les options les plus faibles, qui présentent cependant plusieurs avantages distincts: une communication sans effort et une flexibilité extrême. Il existe de nombreuses façons de dire un mot de passe en toute sécurité à votre homologue. Vous pouvez définir un mot de passe différent pour chaque message si vous le souhaitez.


2. Une clé fixe. L'option forte. Une clé 256 bits complète qui est en outre mutée pour être différente pour chaque message envoyé. Générez un certificat fixe sur un appareil et scannez un QR-code par l'autre.


3. Un certificat unique. L'option la plus solide. Il s'agit d'une clé 256 bits complète à usage unique uniquement. De plus, une clé a un délai d'expiration. L'inconvénient est que pour échanger les certificats uniques dont vous avez besoin pour collecter deux appareils ensemble, il n'y a aucun moyen de le faire à distance.


Alerte nouveaux messages


Il existe une option pour être averti lorsqu'un nouveau message arrive. Cependant, cette fonctionnalité a certaines limites. Nous utilisons une mécanique fournie par le système pour déclencher une vérification du courrier. Par conséquent, nous ne pouvons pas contrôler la fréquence à laquelle il est vérifié, car le système le décide en fonction du modèle d'utilisation de l'application. Si vous lancez régulièrement l'application, le délai est d'environ une à deux heures.


Autres caractéristiques

  • Aucun message électronique n'est conservé sur l'appareil
  • Non se plaint qu'un mot de passe est incorrect - chaque mot de passe vous connecte à un environnement de travail
  • Plusieurs comptes de messagerie avec une boîte de réception combinée
  • Stockage crypté local pour le carnet d'adresses, la galerie et les notes
  • Aucune connexion non sécurisée. Pas de TLS / SSL = pas de connexion
  • Bien que l'objet du message soit crypté, il n'est pas sécurisé, ne mettez pas d'informations sensibles là-bas
